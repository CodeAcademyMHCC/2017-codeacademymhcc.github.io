Nearly all of your programming work — regardless of language or type of project — will be done in some kind of text editor or IDE (Integrated development environment). Here are some worth exploring:

Atom
Very popular, open-source editor created by GitHub. Updated consistently, lots of add-on packages and has good traction in the developer community. Good for many types of development, not just web. It's also an open-source projectl
Sublime Text ()
Another popular text editor that inspired of lot of what came to be in Sublime. It has a very extensive add-on package ecosystem as well. Good for many types of development, not just web.
Brackets
This is a very web-centric text editor by Adobe. It's also an open-source project.
Dreamweaver ()
A robust IDE that's been around for a long time. Designed for devleoping websites and includes tools that let you design visually, not just through code.
Visual Studio Code
An IDE offering from Microsoft designed to work with wide range of languages and types of development. Also free and open-source.
Eclipse
An opens-source IDE also suited for a wide-variety of projects but traditionally very Java-focused. Been around a very long time.
Notepad++ ()
A very light-weight text editor for Windows.
TextMate ( )
A macOS-only text editor that's also been around a long time.
Coda ( )
A web-centric IDE that has good tools for built-in (S)FTP and shell management. The company is based in Portland!
Vim
The terminal-based editor and almost certainly the oldest on this list. Very different than all the other text-editors with a steeper learning curve, but can be very efficient and effective when learned.
