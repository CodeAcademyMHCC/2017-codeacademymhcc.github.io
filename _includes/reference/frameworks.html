A number of these projecst were used in creating this site!

HTML5 Boilerplate
A popular front-end web template that demonstrates the most common practices to assure compatibility across a wide variety of devices. They also provide a neat template generator that lets you pick and choose the components to start your project with.
Bootstrap
A front-end framework that addresses some of the most common needs for the web. View the examples on the Create the Interface page to get a sense.
Font Awesome
A icon-font that provides scalable, vector icons that can be customized and included in your projects.
